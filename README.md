# masbs-xposter

## Dev Env

* Install NodeJS
* run `npm install`
* Copy `user/config.template.json` to `user/config.json`
* Set your mastodon & bluesky configuration in this `user/config.json` file

### Build

* run `npm run build`

### Tests

* run `npm run test`

### Run a sample

* After building run `node ./dist/playground/index.js`
